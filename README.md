# ✨ Glam AI Clone – AI-Powered Beauty & Content Creation Studio

### Transform Photos & Videos with AI-Powered Beauty Filters, Generative Effects, and Creative Editing Tools

---

## 📱 Overview

Glam AI Clone is a modern Android application that brings advanced AI-powered photo and video editing capabilities to mobile users. The app combines intelligent beauty enhancement, generative visual effects, and content creation tools into a sleek and user-friendly experience.

Built using **Kotlin**, **Jetpack Compose**, and **Google Gemini AI**, the application enables users to create professional-quality social media content with minimal effort.

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/Codewith-Arunava/Glam_ai_clone.git

cd Glam_ai_clone
```

### Open Project

```text
Android Studio → Open Existing Project
```

Select the cloned project directory.

---

## 🔑 Environment Variables

Create a `.env` file:

```env
GEMINI_API_KEY=YOUR_API_KEY
```

Obtain your API key from:

[Google AI Studio](https://aistudio.google.com/?utm_source=chatgpt.com)

---

## ▶️ Running the Application

1. Open Android Studio
2. Sync Gradle
3. Add Gemini API Key
4. Connect Emulator or Android Device
5. Click Run ▶️
